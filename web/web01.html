웹 기초 기술

웹은 인터넷을 통해 정보를 주고 받는 시스템
웹 페이지는 HTML, CSS, Javascript 등의 기술로 만들어진다

HTML

HTML은 웹 페이지의 구조를 정의하는 언어이다.
HTML 태그를 사용하여 텍스트, 이미지, 링크 등을 배치할 수 있다.

CSS

CSS는 웹 페이지의 스타일을 정의하는 언어이다.
CSS 사용하여 글꼴, 색상, 레이아웃 등을 조절할 수 있다.

Javascript

Javascript는 웹 페이지의 동적인 기능을 추가하는 프로그램 언어이다.
Javascript 사용하여 사용자와 상호 작용하는 기능을 구현할 수 있다.
